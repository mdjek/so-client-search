# so-client-search

React-приложение позволяет находить вопросы и ответы по запросу с сайта stackoverflow.
С возможностью получать похожие вопросы по автору и тегам.

## Example 

https://mdjek.github.io/so-client-search/

Для запуска проекта:

### `yarn i`
### `yarn start`

Для запуска проекта:

### `yarn build`
